# Flight Reliability & Delay Analysis

An interactive analytics project investigating flight disruptions, cancellations, and performance metrics across major airlines and airports.

## Project Overview
This dashboard processes a dataset of flights to provide actionable insights into aviation reliability. It identifies operational bottlenecks and visualizes the impact of various delay factors.

## Key Analytics & Features
* Disruption Tracking: Monitored cancellations and delays.
* Root Cause Analysis: Categorized cancellations by Weather, Airline/Carrier, and National Air System.
* Performance Benchmarking: Comparative analysis of airlines and airports.
* Time-Series Insights: Monthly trends of average delay times vs. total delayed flight counts.

## Technical Stack
* Tool: Tableau Desktop / Tableau Public
* Visualizations: Dual-axis charts, Treemaps, Heatmaps, and Ranking tables.
* Metrics: Average Delayed Time, Cancellation Rate %, Delay Distribution.

## Preview
The dashboard allows users to filter by specific reasons and timeframes to pinpoint exactly when and why flight performance fluctuates.
